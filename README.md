## Instalasi

Clone from Github

```
git clone https://github.com/adzinpratama/warehouse.git
```

Masuk Ke direktori warehouse

```
cd warehouse
```

install package yang dibutuhkan

```
yarn install
```

start app

```
yarn start
```
